# Preparation

## System Requirements

- Operating System: Linux (64-bit), Windows 10 (limited testing).

## Build from Source  
Use the Go programming language. For official Go documentation:[Installing Go from source](https://go.dev/doc/install/source)

## Install from Package  
In your Linux home directory, create a folder named `dsysb`.  
Place the three downloaded executable files into this folder.  
The three documentation of them:  
- [__dsysb__](dsysb.md).
- [__dsysbminer__](dsysbminer.md).
- [__dsysbcmd__](dsysbcmd.md).
