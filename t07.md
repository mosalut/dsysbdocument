# transfer

_transfer_ refers to a transfer transaction.

## Fields

- _from_: sender's wallet address.
- _to_: Recipient's wallet address.
- _hier_: Heir's wallet address.
- _amount_: Amount.
- _assetId_: The asset ID to be transferred. It will be all 0s if left blank, and will also represent _DSB_.
- _nonce_: Preventing replay attacks.
- _bytePrice_: Byte price.
- _signer_: Signature.

The following is a complete _transfer_ transaction command:

```bash
$ ./dsysbcmd create transfer '{"from":"DCUz2Z2D8C9YC7ZWaVBwAx16vygpAG4fST","to":"DUAv3s7ATF1s3ghkzPGDCTSFdgSTEfJw6g","amount":300000000,"bytePrice":1}'
```

After a transfer, you can use the _getbalance_ or _getaccount_ commands to check your wallet balance or account activity.
